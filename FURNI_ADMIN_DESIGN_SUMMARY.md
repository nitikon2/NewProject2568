## 🎭 Animations & Effects

### ⚡ Keyframe Animations
- **furniFloat**: Floating effect (6s infinite)
- **furniPulse**: Pulsing effect (3s infinite)  
- **furniShimmer**: Shimmer effect (3s infinite)
- **furniGlow**: Glowing effect (3s infinite)
- **furniWave**: Wave morphing (8s infinite)

### 🎨 CSS Classes
- `.furni-hover-lift`: Hover lift effect
- `.furni-glass`: Glass morphism
- `.furni-btn`: Enhanced button styles
- `.furni-card`: Modern card effects
- `.furni-gradient-text`: Gradient text

### 🌈 Color Scheme
```css
:root {
  --furni-primary: #2f4b3f;
  --furni-gold: #f9c74f;
  --furni-bg-primary: #f7f5f3;
  --furni-glass-bg: rgba(255, 255, 255, 0.9);
}
```
